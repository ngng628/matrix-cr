# matrix-cr

Implements Ruby's Matrix library.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     matrix-cr:
       github: ngng628/matrix-cr
   ```

2. Run `shards install`

## Usage

```crystal
require "matrix"
```
